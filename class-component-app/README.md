## Rick and Morty Character Search

This project is a single-page React application for browsing Rick and Morty characters. It provides a searchable, paginated list of characters and a details panel for the currently selected character, all wrapped in a clean layout with header, footer, and theme toggle support.

The app is built with **React + TypeScript**, **Redux Toolkit** (RTK Query) for data fetching and caching, **React Router** for navigation and URL query parameters, and **Vite** as the build tool.
